COME PUBBLICARE GRATIS SU GITHUB PAGES

1. Vai su https://github.com
2. Crea un nuovo repository chiamato: vaulta
3. Scegli repository Public
4. Carica tutti i file di questa cartella
5. Vai su Settings > Pages
6. In Source scegli:
   - Branch: main
   - Folder: /root
7. Salva
8. Dopo qualche minuto il sito sarà online a un indirizzo tipo:
   https://tuo-utente.github.io/vaulta/

COSE DA MODIFICARE

1. Link acquisto:
Nel file index.html cerca:
https://payhip.com/
Sostituiscilo con il link del tuo prodotto Payhip.

2. Email:
Nel footer cerca:
tua-email@example.com
Sostituiscila con la tua email commerciale.

3. Screenshot:
Quando avrai screenshot reali di Vaulta, possiamo aggiungerli in una sezione dedicata.
